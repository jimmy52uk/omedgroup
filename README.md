
# OMED Group Website (Next.js + Tailwind)

This repository contains a production-ready Next.js scaffold for the OMED Group website.
Includes Tailwind CSS, framer-motion for animations, and Lucide icons.

## Run locally

1. Install dependencies:
   npm install

2. Run dev server:
   npm run dev

3. Build:
   npm run build

## Deploy

Recommended: Vercel (import GitHub repo and deploy)
